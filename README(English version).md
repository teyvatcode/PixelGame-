This is my first project called PixelGame.

Based on Cursor and Copilot, i finally made it to some extend.
This is a game with 20 levels and player use four direction keys to help the character reach the green block(the end).

Players can upgrade their character and choose different weapons during the game.
Since all the characters, maps... the whole game is made by AI, there are some place which players cannot reach directly.
But, there still exists a way to reach these places.It needs you to explore.

It's a pity that i donnot know how to make a balance in this game(including how many gold coins can a player get in one game, how many blood volume the monster have should be better). It's disappointed.

这是我的第一个项目PixelGame。

基于Cursor和Copilot实现，我在一定程度上完成了这个游戏。

整个游戏有20关，玩家使用四个方向键来帮助角色到达绿色的终点区域。

玩家可以在游戏过程中提高角色和武器的等级。

由于整个项目都是依靠AI实现的，所以有些区域可能并不能直接抵达。但是仍然有别的方法可以抵达这些区域，这需要你自己进行探索。

很遗憾的是我并不知道如何调整游戏的平衡性（这里包括了金币的获取，怪物的血量等内容）。
